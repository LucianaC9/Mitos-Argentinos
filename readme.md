#  Catálogo de Mitos Argentinos

Este proyecto fue realizado para el curso de Python Intermedio del curso Tecno3F.
 Es Una aplicación web desarrollada en **Flask** que permite al usuario registrar, visualizar, editar y eliminar mitos tradicionales de Argentina. La app cuenta con una interfaz moderna y fácil de usar, respetando un diseño con colores pastel, tipografía moderna e íconos representativos, incluyendo el logo del mapa argentino.

---

## Objetivos del Proyecto 🎯
- Implementar un sistema CRUD sobre cualquier temática
- Que contenga mínimo 3 tablas relacionadas con DDBB SQLite
- Que contenga mínimo 5 campos de entrada (input, select, check, etc)



## Funcionamiento de la Aplicación 💻

- El usuario accede a la pantalla principal para **cargar nuevos mitos argentinos**.
- Se completan los siguientes campos:
  - ✅ Nombre del mito
  - ✅ Descripción
  - ✅ Región (selección de lista)
  - ✅ Tipo de mito (selección de lista)
  - ✅ Ruta o URL de una imagen representativa
- El usuario puede:
  - ✔ Agregar un mito
  - 👁 Ver el listado completo
  - ✏ Editar o 🗑 eliminar mitos existentes
- Los mitos se muestran en una tabla y botones de acción.
- Toda la información se almacena en una base de datos local `database.db`, que se genera automáticamente al iniciar la app.
